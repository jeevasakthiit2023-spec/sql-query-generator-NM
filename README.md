# sql-query-generator-NM :
SQL Query Generator is an AI-powered application that converts natural language queries into SQL queries. The system uses the IBM Granite 3.3 2B Instruct model to understand user input and automatically generate accurate SQLite SQL queries.

This project is developed using Python, Gradio, SQLite, and Machine Learning libraries and runs on Google Colab. Users can simply type requests such as “Show users who signed up last month” and the system generates the corresponding SQL query.

The platform also includes security validation to prevent SQL injection, intelligent table and column detection, and export features like PDF reports and text files. With its chat-based interface, the system makes database querying easy even for users without SQL knowledge.
